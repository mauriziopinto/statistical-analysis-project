# Statistical Analysis — Medical Insurance Costs

This project conducts a statistical analysis of US medical insurance costs using descriptive statistics, visualizations, and hypothesis testing. It examines the relationship between smoking status and insurance charges using the Insurance dataset from Kaggle.

**Dataset:** Insurance (Medical Insurance Costs) — https://www.kaggle.com/datasets/mirichoi0218/insurance

## How to Run

1. Clone this repository
2. Set up the environment:
   ```bash
   uv sync
   ```
3. Open and run the notebook:
   ```bash
   uv run jupyter notebook analysis.ipynb
   ```
4. Alternatively:
   ```bash
   pip install -r requirements.txt
   jupyter notebook analysis.ipynb
   ```
